# Agent X - 下一代智能体平台/协议品牌站

agentx.md 官方网站，展示下一代智能体平台、协议和创新项目。

## 功能模块

- **项目展示**：精选创新型智能体应用、协议、工具项目展示
- **开发资源**：API接口、开发框架、开发者工具、学习资源汇总
- **行业公告**：最新Agent领域融资事件、创业动态、技术进展

## 技术栈

- React 18 + Vite
- React Router 6
- 响应式设计，支持移动端

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

## 部署

推荐部署到 Vercel 或 Netlify，支持自动部署和SSL证书。
